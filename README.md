# Big Data Job Market Pipeline

Complete Big Data pipeline for job market analytics using Apache Kafka, Hadoop HDFS, Apache Spark, Apache Hive, Docker, and Power BI.

## Technologies Used
- Apache Kafka
- Hadoop HDFS
- Apache Spark
- Apache Hive
- Docker
- Power BI

## Project Files
- `Proposal_job.pdf`: Project proposal
- `screenshots/`: Dashboard and pipeline screenshots
- `Excel files`: Job postings dataset
